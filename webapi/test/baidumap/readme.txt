
功能说明:
    1.获得当前地理坐标的经纬度
    2.判断当前位置是否在指定目标的经纬度区域范围内
使用说明
    该功能依赖百度地图实现
  1.在HTML中首先要引用百度地图和jquery库
        api.map.baidu.j
        developer.baidu.com.js
        api.map.GeoUtils.js
        
     这几个库文件存放在:baidumap/map/目录下
  2.在HTML中引用 baidumap.js
      这个文件存放在 baidumap/目录下，主要实现了
      apiGetLocalPosition 和apiIsPointInCircle 两个API，具体实现和说明请看apiBaiduMap.js中函数注释说明
  3.具体使用例子请参见 test/baidumap/目录下 demo.html demo.js的具体实现
    demo.html中通过点击事件调用 JS函数 test()；
    demo.js 是 test()的具体实现。
    需要注意的是：test()的业务逻辑是需要根据具体业务来决定。
